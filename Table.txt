Core Structure of a Theta Table

A Theta table is built from:

Boards → the main workspace/table
Groups → sections inside the table
Items → rows/tasks/records
Columns → fields/data types
Subitems → nested tasks under a row
Views → different visual formats of the same data
1. Rows / Items Features ✅

Each row (called an “Item”) can contain:

Task/project/client name
Assignees
