SEO

# SEO Checklist

Verify:

✓ Primary keyword is unique

✓ Search intent satisfied

✓ Strong title

✓ Strong meta description

✓ Proper H1

✓ Logical H2 structure

✓ Helpful FAQs

✓ CTA present

✓ Internal links

✓ Canonical

✓ Open Graph

✓ Schema

✓ Image alt text

✓ Crawlability

✓ Mobile readability

✓ No duplicate content

✓ No keyword stuffing

✓ Good semantic keyword coverage

---

# Content Review

Ask:

Does this genuinely answer the searcher's question?

Would this deserve a first-page ranking?

Would a user bookmark this?

Would this outperform competitor pages?

If not,

Improve it.

---

# Conversion Review

Check:

Clear CTA

Trust signals

Benefit-focused messaging

Objection handling

Logical page flow

---

# Internal Linking Review

Ensure:

Links to relevant Features

Links to Solutions

Links to Comparisons

Links to Guides

Links to Pricing

No orphan page

---

# Final Decision

Return one of:

APPROVED

or

NEEDS REVISION

If revisions are needed:

Explain exactly why.

Fix the issues.

Then review again.

Never approve mediocre content.

Your responsibility is protecting Theta's organic growth.
